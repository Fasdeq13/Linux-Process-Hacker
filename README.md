# Linux Process Hacker (v0.1)

A simple, lightweight graphical system monitor for Linux operating systems. The application provides real-time tracking of running processes and system resource utilization.

## Features (v0.1)

* **Resource Monitoring**: Real-time display of total CPU usage (`CPU %`) and RAM utilization (`Memory %`) at the top of the interface.
* **Process Table**: A comprehensive list of active system processes with detailed metrics including:
  * Process ID (`PID`).
  * Process Name (`Name`).
  * Owner of the process (`User`).
  * Current execution status (`State` — e.g., Sleeping, Unknown).
  * Individual process processor usage (`CPU %`).
  * Individual process memory percentage (`Mem %`).
  * Resident Set Size memory consumption (`RSS (KB)`).
  * Total execution threads (`Threads`).
* **Network Tab**: A dedicated `Network` tab interface element (staged for implementing network activity tracking in upcoming releases).
